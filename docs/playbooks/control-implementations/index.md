## How to Use These Playbooks

1. **Start with Portal Walkthrough** to understand what needs to be configured and where
2. **Use PowerShell Setup** to automate configuration at scale and create repeatable deployments
3. **Run Verification and Testing** to validate controls are working and collect evidence
4. **Reference Troubleshooting** when issues arise during implementation or operation

Each playbook is self-contained but references related playbooks within the same control and across related controls.
