# Playbooks

Implementation procedures for the FSI Copilot Governance Framework. Each playbook provides step-by-step guidance for implementing, validating, and maintaining governance controls for Microsoft 365 Copilot.

---

## Control Implementation Playbooks

Every control in the framework has four dedicated playbooks:

| Playbook | Purpose |
|----------|---------|
| **Portal Walkthrough** | Step-by-step admin portal configuration with screenshots and navigation paths |
| **PowerShell Setup** | Automation scripts for configuration, bulk operations, and reporting |
| **Verification & Testing** | Test cases, evidence collection, and compliance validation procedures |
| **Troubleshooting** | Common issues, diagnostic steps, and resolution guides |

- [Pillar 1 — Readiness & Assessment](control-implementations/index.md#pillar-1-readiness-and-assessment-13-controls) (13 controls, 52 playbooks)
